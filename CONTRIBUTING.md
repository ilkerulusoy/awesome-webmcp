# Contributing to Awesome WebMCP

Thank you for helping keep this list accurate and up to date!

## How to add a resource

1. **Fork** this repository and create a new branch.
2. **Edit** `README.md` — add your link in the most relevant category.
3. **Open a Pull Request** with a short description of what you're adding and why it belongs here.

## Quality criteria

A resource is a good fit if it meets **all** of the following:

- Directly related to WebMCP / `navigator.modelContext`
- Publicly accessible (no paywalls or login walls)
- In English (non-English resources are accepted if they provide unique value not covered elsewhere)
- Not purely promotional or advertisement-driven

## Link format

Follow the existing style exactly:

```markdown
- [Title – Source](https://example.com) — One-sentence description of what the reader will find.
```

- Use an em dash (`—`) before the description, not a hyphen.
- Keep descriptions to **one sentence**.
- Place the link in the correct category. If no category fits, suggest a new one in your PR.

## Categories

| Category | What belongs here |
|---|---|
| Official Resources | Specs, official Chrome blog posts, official extensions |
| Specification & Explainers | Deep-dives into the API surface by spec authors or browser engineers |
| SDKs & Libraries | npm packages, polyfills, TypeScript types, framework integrations |
| Chrome Extensions & DevTools | Browser extensions and DevTools panels for WebMCP |
| Demos & Examples | Runnable demos, starter repos, playgrounds |
| Tutorials & Guides | Step-by-step guides and how-to articles |
| Videos | Screencasts, conference talks, explainer videos |
| Articles & Blog Posts | News coverage, opinion pieces, technical analyses |
| X (Twitter) Threads | Noteworthy threads from developers or browser vendors |
| Tools & Utilities | Online checkers, generators, debugging tools |

## What we don't accept

- Duplicate links already in the list
- Links that are broken or return errors
- Resources unrelated to WebMCP (general MCP server lists, AI agent frameworks without WebMCP integration, etc.)
- AI-generated articles with no original technical content

## Updating or removing a link

If a link is broken or a resource is outdated, open an issue or PR explaining the problem.
